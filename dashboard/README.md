# Swarm Dashboards

Dashboards copies en direct depuis les repos externes :

- `swarm-ui/` : transplant direct de `MiroShark/frontend`, avec `npm run run` comme alias de lancement et `npm run help` pour l'aide Vite
- `swarm-ui-alt/` : transplant direct de `AI-Swarm/frontend`

Points d'entree :

- `index.html` : landing page locale
- `swarm-ui/index.html`
- `swarm-ui-alt/index.html`

Ces frontends restent presque tels quels. Ils servent de base UI et de reservoir de patterns frontend pour les futures surfaces canoniques du repo.
